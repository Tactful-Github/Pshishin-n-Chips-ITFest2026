# Pshishin' Quest
This project was made for ITFest 2026.

Phishin' Quest is an application that allows its users to learn the basis of cybersecurity through gamified lessons.


## Context

You are an employee at a busy IT company. After a long day of work, you receive a suspicious email. Exhausted and distracted, you mistakenly click the embedded link and your computer is instantly hacked!

Can you stop the virus from spreading through the office before your boss finds out? Decrypt as many computers as you can, learn new skills, and have fun with our game!


## Educational Track

- **Active Defense:** This game was designed with the intention of moving players from passive learning to active defensive thinking
- **Target Audience:** Middle schoolers looking for an engaging entry point into the world of cybersecurity



## Gameplay Features

**Modular cubicle system:** Each cubicle represents a unique challenge that the users must solve

**Hacker Toolkit:** The users have interactive in-game tools that help them complete the levels (magnifying glass, phone etc.)

**Reward System:** Players earn money for each round they solve; these coins act not only as a score, but a gamified performance metric


## Learning Path [so far]
- Phishing
- Web
- Crypto



## Tech Stack

**Engine:** Unity 6.3 (LTS)

**Environment:** 2D Sprites integrated in a 3D Ortographic workspace
